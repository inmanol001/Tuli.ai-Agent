# Skill: browser_search

## Objetivo

Usa browser_search cuando el usuario quiera buscar información, abrir una página o ver contenido web usando el navegador por defecto.

browser_search puede abrir:

- búsquedas visibles
- Google
- YouTube
- URLs
- sitios conocidos
- páginas web

## Reglas

- Usa browser_search si el usuario quiere ver algo abierto en el navegador.
- Usa browser_search si el usuario dice abrir.
- Usa browser_search si el usuario dice buscar.
- Usa browser_search si el usuario dice navegar.
- Usa browser_search si el usuario dice visitar.
- Usa browser_search si el usuario quiere YouTube visible.
- Usa browser_search si el usuario quiere Google visible.
- Usa browser_search si el usuario da una URL para abrir.
